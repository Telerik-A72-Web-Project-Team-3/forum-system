About CineTalk Forum

Learn more about the project and the people behind it.

What is this project?

CineTalk is a discussion forum focused on movies and series. It is built as a full-stack Spring Boot application with a custom forum engine, folder hierarchy, tags, likes, comments, and a clean, modern UI.

Goals

  • Provide a structured place to discuss movies and TV series.
  • Practice clean layering with controllers, services, repositories, and DTOs.
  • Explore modern Spring Security, validation, and pagination patterns.
  • Offer a solid base for future features like notifications and search.

Tech stack

  • Java & Spring Boot
  • Thymeleaf templates with a shared layout system
  • REST APIs, DTO mapping, and validation
  • Relational database with realistic seeded data
